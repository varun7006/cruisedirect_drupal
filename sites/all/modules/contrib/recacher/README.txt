Recacher tracks URLs expired by Cache Expiration (expire) and subsequently warms their cache using HTTPRL.

Drupal project page is at https://www.drupal.org/project/recacher
Installation, demo and tips are at http://www.vacilando.org/article/recacher
If you encounter a problem or need support, see https://www.drupal.org/project/issues/recacher
